## CodeAlpha_HotelReservationSystem (Java)
Java console-based Hotel Reservation System using OOP and File I/O

## Description
Console-based Java application that manages hotel bookings using OOP and File I/O.

## Features
- View available rooms
- Make bookings with payment simulation
- Cancel bookings
- View all bookings

## How to Run
1. Compile: javac HotelReservationSystem.java
2. Run: java HotelReservationSystem
